Autoresearch campaign viewer

Results of prompt experiments. Each battery is one prompt variant run over the same set of labeled test cases, several times per case.

How to read this page
  • Every test case has a known correct answer decided beforehand (the label).
  • Each case was asked several times (runs). The model's answer is the majority across those runs; if the runs disagreed, the case is marked unstable.
  • A case counts as ✓ correct when the model's majority answer matches one of the case's accepted answers, and ✗ wrong otherwise.
  • Click any case row to fold open the individual runs: the answer given, the model's own explanation, response time, and the raw output.
  • Load two or more batteries to compare prompt variants. The first battery loaded is treated as the baseline; comparison rows where variants answered differently are highlighted.
  • Tags group cases (by source dataset, failure mode, or provenance) so you can see where a variant helps or hurts.
